# Coursera Dump

Скрипт выбирает 20 случайных курсов на Coursera, получает информацию о них (название, язык, ближайшую дату начала, количество недель и среднюю оценку), и записывает её в xlsx-файл, имя которого передаётся в качестве аргумента.

# Quickstart

Скрипт требует для своей работы установленный интерпретатор Python версии 3.5, а также некоторые дополнительные зависимости, которые можно установить следующим образом:
```
pip install -r requirements.txt
```

Запуск на Linux:

```#!bash
$ python coursera.py courses.xlsx
```
Запуск на Windows происходит аналогично.
